Introduction

The purpose of this R shiny application allows US residents to track monkeypox/orthopoxvirus cases via interactive map and statistical plots.

Summary

* This R-shiny application focuses on producing a chronograph map and a time series plot to visualize monkeypox/orthopoxvirus cases in the United States (US) as a web application, developed using the Shiny and Leaflet library in R.

* Three small boxes on the top of the dashboard display real-time cases information such as, "Total Confirmed Cases", "Daily Cases", and "Last Update", respectively.

* An interactive US chronograph map to display confirmed cases in each state. The map can either display confirmed cases for each state by zooming in with scroller or obtain aggregated confirmed cases by zooming out with scroller.

* Provides a time series bar plot for users to observe the confirmed cases change in a metric over time as a series of bars. (read more)
