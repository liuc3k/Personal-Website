Introduction

A R-shiny application for users not only to track Covid-19 cases/deaths in the US map or certain states but also acquire related demographic plots/tables interactively.

Summary

* Covid-19-DataViz-in-US is a user-friendly R-shiny application for exploring Covid-19 cases/deaths data in the United State (US) via interactive chronograph map and several statistical plots.

* Allow users to obtain real-time information about Covid-19 confirmed/cumulative cases and deaths by interacting with the chronograph map at state level or at the county level under a specified state.

* Track the movement of the Covid-19 cases/deaths via time series plots such as line plots and time series bar plots

* Obtain some ideas regarding demographic characteristics such as race, sex, and age group,etc by using bar charts. (read more)
